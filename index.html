Apex is an API proxy for microservices. Designed for small teams that have just begun migrating from a monolith to a microservices architecture, it provides one place to log and control service-to-service traffic. Apex addresses several challenges introduced when services communicate over the network, such as debugging with distributed logs, duplicate logic in client libraries for API management, and defining and managing global and service-to-service configuration.
